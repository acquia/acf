INTRODUCTION
------------

ACF Product Search provides configuration for Search API and Facets to use
with ACF Product.


REQUIREMENTS
------------

This module requires the following modules:

 * ACF Product (https://github.com/acquia/acf)
 * Facets (https://www.drupal.org/project/facets)
 * Facets Summary (https://www.drupal.org/project/facets)
 * Search API (https://www.drupal.org/project/search_api)
 * Search API DB (https://www.drupal.org/project/search_api)


INSTALLATION
------------

 * Install as you would normally install a contributed Drupal module.
   See: https://www.drupal.org/node/895232 for further information.


CONFIGURATION
-------------

No configuration is needed.
